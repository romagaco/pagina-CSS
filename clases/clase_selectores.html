Selectores CSS

Selectores: identificadores de elementos dentro de una página web

Los selectores en CSS son patrones que indican al navegador qué elementos HTML se van a seleccionar para aplicarles estilos. Son la primera parte de una regla CSS.

  • Selector universal, este seleccionara todos elementos HTML 
                
              * {
                  color: red;
                }
  • Selector por etiqueta, este aplicara los atributos unicamente a las etiquetas, ejemplo el color azul se aplicara unicamente a los parrafos y el verde a los elementos de las listas 
     
            
              p {
                  color: blue;
                }
              li {
                  color: green;
                }
  • Selector por clase, este aplicara los atributos a quellas etiquetas que sean definidas como clase ejemplo a "destacado" se le aplica un color naranja y un fondo oscuro, a "salmon" un color rosa. 
                
              .destacado {
                  color: orange;
                  background-color: black;
                }
                
              .salmon {
                  color: pink;
                }
  • Selector por multiclase si una etiqueta se define como dos clases distintas, esta recibe los atributos de ambas calses, ademas podemos otorgar atributos unicamente a las etiquetas que posean ambos tipos de clases. 
                
              .destacado.salmon {
                font-size: 1.2rem;
                border: 5px solid rgb(211, 211, 57);
              }
  • Selector por id, se define un id, este sera unico por html, se llama utilizando el simbolo # 
                
              #punk {
                color: purple;
              }
  • Selector mixto, nos permite combinar una etiqueta + clase, un id + clase, etc 
                
              h4.Olive {
                color: Olive;
                }
  • Selector por atributos, entre corchetes [] colocamos lo especifico y por fuera que tipo de elemento buscar en el ejemplo buscamos cualqueir elemento que se defina como una multiclase destacado punk. 
                
              * [class="destacado punk"] {
                font-size: 0.5rem;
                }
